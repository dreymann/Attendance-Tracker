To compile and run the program from console traverse to AttendanceTracker\src path and run:

javac Attendance.java
javac AttendanceController.java
javac Course.java
javac GraphingData.java
javac HomeScreen.java
javac ReadData.java
javac RecordData.java
javac Student.java
javac StudentToCourse.java



java AttendanceController


++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++
